# 📡 Signal Lost
**Exploração Procedural · Sobrevivência · Resgate**
*UNIDADE-7 acordou em um mundo desconhecido. Ative as antenas. Envie o sinal. Aguarde o resgate.*
**Signal Lost** é um jogo de sobrevivência em HTML5/Canvas focado em exploração procedural, gerenciamento de recursos e combate contra hordas. Lute para sobreviver em diferentes biomas, colete minérios, construa defesas, evolua suas habilidades e ative antenas para chamar a nave de resgate.

## 🚀 Como Jogar
O objetivo principal no modo padrão é **encontrar e ativar as 5 antenas de rádio** espalhadas pelo mapa e sobreviver até a chegada da nave de resgate.
### 🎮 Controles Principais
 * **WASD** ou **Setas**: Movimentação
 * **Mouse**: Mirar
 * **Clique Esquerdo**: Atirar / Construir / Interagir
 * **Botão Direito** ou **B**: Mudar bloco de construção
 * **1 a 7**: Selecionar armas (desbloqueadas por nível)
 * **Q**: Equipar Laser (Arma)
 * **E**: Equipar Ferramenta de Construção
 * **R**: Equipar Ferramenta de Destruição
### 🎒 Inventário e Sobrevivência
 * **I**: Abrir Inventário & Crafting
 * **C**: Fabricar item (com inventário aberto)
 * **T**: Usar item selecionado
 * **X**: Descartar 1 item sob o mouse (Shift + X para descartar a pilha toda)
 * **G**: Scanner (Segure sobre um bloco próximo para catalogar)
### ⚡ Habilidades e Especial
 * **U**: Abrir Árvore de Evolução (Upgrades)
 * **F**: Teleporte de emergência até o cursor (Gasta toda a energia, 4s de recarga)
 * **V**: Navegação A.R.I.A (Cria uma rota holográfica até a antena mais próxima)
## ✨ Características do Jogo
 * **Mundos Procedurais:** Geração de mapas baseada em *Seeds*, garantindo que cada partida seja única.
 * **Múltiplas Dimensões:** Explore a Superfície, o Subsolo (Cavernas) e a misteriosa Dimensão Void através de portais.
 * **Sistema de Biomas:** Água, Lava, Neve, Pântano, etc., afetando a física e a sobrevivência do jogador.
 * **Construção de Base:** Crie barreiras, paredes de cristal e armadilhas para se defender das hordas noturnas.
 * **Árvore de Evolução (RPG):** Ganhe XP matando inimigos e suba de nível para gastar pontos em Vida, Dano, Velocidade, Regeneração e muito mais.
 * **Modos de Jogo:** Finito (Resgate), Infinito e Criativo (Construção livre e sem dano).
## 💻 Como rodar o projeto localmente
Este jogo é puramente construído em HTML, CSS e JavaScript Vanilla (sem frameworks).
 1. Faça o download do código.
 2. Não é necessário compilar (build) nada.
 3. Para jogar, basta abrir o arquivo index.html em qualquer navegador moderno (Chrome, Firefox, Edge).
   *(Nota: dependendo de algumas funções de import/export ou módulos locais, pode ser recomendável usar uma extensão como o "Live Server" do VS Code para rodar via http://localhost ao invés de file://).*
